A WordPress plugin to fetch customer license data from Freemius. 
 
== Description ==
 
Fetching data from Freemius and sending to SureCart.
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
